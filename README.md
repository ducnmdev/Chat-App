### Clone the repository

```shell
git clone https://github.com/ducnmdev/Chat-App.git
```

### Install dependencies

```shell
npm i
```

### Setup .env file

```shell
MONGODB_URI=...
PORT=5001
JWT_SECRET=...

CLOUDINARY_CLOUD_NAME=...
CLOUDINARY_API_KEY=...
CLOUDINARY_API_SECRET=...

NODE_ENV=development
```

### Start the app

```shell
npm run dev
```
